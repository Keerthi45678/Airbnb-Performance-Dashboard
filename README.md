# Airbnb Performance Dashboard

A simple and interactive dashboard project focused on understanding Airbnb trends through data.

---

### **1. Project Description**

Analyzed Airbnb data to identify patterns in listings, reviews, and ratings using Power BI dashboards.

---

### **2. Tech Stack**

* Power BI
* Excel

---

### **3. Data Source**

* Dataset prepared and structured using Excel for analysis purposes

---

### **4. Features / Highlights**

* 📊 Overview dashboard for key metrics and listing distribution
* 💬 Reviews analysis to understand user engagement
* ⭐ Ratings insights across different locations
* ⚡ DAX measures for calculated metrics and KPIs
